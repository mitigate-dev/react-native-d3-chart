# react-native-d3-chart

Create performant charts with zooming, panning and localization

## Installation

```sh
npm install react-native-d3-chart
```

## Usage


```js
import { D3ChartView } from "react-native-d3-chart";

// ...

<D3ChartView color="tomato" />
```
